# Tableau-DB
This project is an interactive sales dashboard built using Tableau to analyze and compare business performance across different regions.
   
1. Project Title / Headline

    A concise, descriptive name for the dashboard. Region-Based Sales Comparison Dashboard
    The dashboard provides a side-by-side comparison of Primary Region and Secondary Region, enabling users to evaluate sales performance, profitability, and             customer behavior efficiently.

 2. Short description / Purpose 
   1-2 sentences explaining what the dashboard does and why it exists.
   
   Example: 
   📊 Region-Based Sales Comparison Dashboard
   
   This dashboard, built using Tableau, compares sales performance across different regions by visualizing key metrics like sales, profit, and customer trends.
   Its purpose is to help stakeholders quickly identify high-performing regions, analyze patterns, and make data-driven business decisions.

 3. Tech Stack
    List the key technologies used to buid the dashboard.
    Example:
    The dashboard was built using the following tools and technologies:
       * Tableau (Tableau Public/Desktop) – Dashboard design & data visualization
       * Microsoft Excel / CSV – Data source & preprocessing
       * Data Visualization Techniques – KPI cards, maps, trend lines, and filters for analysis
       * Tableau Public (Dashboard creation & visualization)
       * Dataset: Retail Sales (Sample Superstore-type data)
         
 4. Data Source
    More info on where the data comes from and how it's structured
    Example:
    
    The data used in this dashboard is sourced from a retail dataset (similar to the Sample Superstore dataset) and stored in Microsoft Excel / CSV format. It is      structured in a tabular form with key fields such as Order Date, Region, Category, Sub-Category, Sales, Profit, Customer ID, and Order ID, enabling detailed       analysis across different dimensions like time, geography, and product segments.

5. Features/Highlights

   The best dashboard explanation format.
    *Region Comparison View:
     Side-by-side comparison of Primary and Secondary regions to easily evaluate performance differences.
   
    *Key KPI Indicators:
     Displays important metrics such as Sales, Profit, Orders, Customers, and Returns for quick insights.
   
    *Interactive Map Visualization:
     Geographic representation of sales distribution across regions for better spatial understanding.
   
    *Category & Sub-Category Analysis:
     Detailed breakdown of product performance with visual comparisons across regions.
   
    *Trend Analysis:
     Time-based charts to track sales and profit trends over a period.
   
    *Dynamic Filters:
     Allows users to select and compare different regions interactively.
   
    *User-Friendly Layout:
     Clean and intuitive dashboard design built using Tableau for easy navigation and analysis.
