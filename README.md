# Tableau-DB
This project is an interactive sales dashboard built using Tableau to analyze and compare business performance across different regions.
   
   1. Project Title / Headline

    A concise, descriptive name for the dashboard. Region-Based Sales Comparison Dashboard
    The dashboard provides a side-by-side comparison of Primary Region and Secondary Region, enabling users to evaluate sales performance, profitability, and             customer behavior efficiently.

    2.Short description / Purpose 1-2 sentences explaining what the dashboard does and why it exists.

    This dashboard, built using Tableau, compares sales performance across different regions by visualizing key metrics like sales, profit, and customer trends.
    Its purpose is to help stakeholders quickly identify high-performing regions, analyze patterns, and make data-driven business decisions.
